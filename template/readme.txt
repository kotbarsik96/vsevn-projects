Для использования шаблона нужно скопировать следующие папки и файлы:
src, gulpfile.js, package.json
Далее запустить установку npm install в папке, куда были скопированы эти файлы и папки

gulpfile.js:

1. Для отключения обфускации перейти в функцию javascripts() и закомментировать .pipe(obfuscator).

2.Для того, чтобы добавлять иконки в иконочный шрифт, нужно взять из /src/fonts/IconFont/ файл icon-font.svg, затем его импортировать в сет на сайте https://icomoon.io/app/#/select. 
После импорта, нужно выделить нужные иконки в сете и нажать "Generate Font", далее "Download", далее открыть архив, из папки fonts перетащить новые файлы в папку /src/fonts/IconFont/, если название отличается от "icon-font.*", поменять его на это название. Далее в архиве открыть файл style.css, из него скопировать все .icon-*::before классы в файл /src/css/fonts.css
